What is this website ?

This organisation aims to simplify the development and deployment of projects made within the Epitech school.

While making C/C++ projects can be easy, or hard, we all know that writing documentation by hand is annoying (At least it's our opinion).
This is why we made this "tool".
The idea was to have an automated way to :

  • Check for errors in the code itself
  • Check for logic errors (using unit testing)
  • Check for coding-style errors
  • Build technical documentation
  • Get a report directly on Discord
  • If (and only if) everything went good, mirror the project to the actual school repository

I'me glad to say that this is now possible using Github Actions
Each time a modification is made on the main branch of a project, a Github Action is ran, which does all the above.
This website is the place you will be able to find all the documentations for these projects.

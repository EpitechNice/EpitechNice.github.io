    cmp

    bool cmp(void *, void *)

    Erases an element from the linked list.

    Parameter head
    A pointer to the head of the linked list.
    Parameter ref
    A comparative value to find the element to erase.
    Parameter cmp
    A function that takes two void pointers as arguments and returns a boolean. The first argument is the given ref and the second is the data of the current element.
    Parameter del
    A function that takes a void pointer as an argument and returns void. Or NULL if no function is needed.

    Warning

    The function assumes that the given list is valid with a NULL at the end. If this is not the case, the behavior is undefined.

    m_list_add_at_pos

    void m_list_add_at_pos(mlist_t *ref, mlist_t obj, mlist_pos_t pos)

    Set a new node to a specific position in a bigger list

    Parameter ref
    Address of the ref list
    Parameter obj
    New node to add (see m_list_new)
    Parameter pos
    M_LIST_FIRST, M_LIST_BEFORE, M_LIST_AFTER, M_LIST_LAST

    Note

    The 3rd argument is using the ref argument as mark : M_LIST_BEFORE means just before the reference

    m_list_apply

    void m_list_apply(mlist_t *ref, void (*func)(void *))

    Apply a function to all members of a list

    Parameter ref
    A reference to the list (does not need to be the beggining)
    Parameter func
    Function to be called on each elemnt of the list.

    m_list_destroy

    void m_list_destroy(mlist_t *ref, void (*destructor)(void *))

    Clear out a mlist

    Parameter ref
    A reference to the list to be cleared (does not need to be the beggining)
    Parameter destructor
    If not NULL, it will be called on each element.

    m_list_find

    mlist_t m_list_find(mlist_t *ref, void *data)

    Find an element in the list

    Parameter ref
    The list to be searched
    Parameter data
    Data to be found on the list
    Return
    The node containing the data, or NULL if not found

    m_list_get

    void *m_list_get(mlist_t *ref, size_t pos)

    Get the value at a given position (it's operator[] ))

    Parameter ref
    The list to be used
    Parameter pos
    The index to get the value
    Return
    The value found at given position, or NULL if there where an error

    m_list_new

    mlist_t m_list_new(void *data)

    Build new mlist node containing the provided data

    Parameter data
    Pointer to object to save (char array, struct...)
    Return
    Newly built mlist node

    Tip

    Remember ! You can stack function together ! So, saying m_list_add_at_pos(ref, m_list_new(ptr), MLIST_POS_LAST) is 100% valid ! Until we meet again !

    m_list_pop

    void *m_list_pop(mlist_t obj)

    Remove the passed argument of the list he was in,

    Parameter obj
    The node to be deleted
    Return
    The data that were in the object

    Warning

    It does NOT work like the python pop list method. You need to give an element of the list, not an index.
